Presentation du projet

EasyClubSport
Laravel API Vue 3 Tailwind CSS

Notre idée

Centraliser toute la gestion d'un club sportif dans une seule application

EasyClubSport aide un club à organiser ses équipes, gérer ses joueurs, suivre les événements, simplifier la communication et rendre l'organisation plus professionnelle.

Besoin réel Projet complet Architecture moderne

Message principal

Cette idée est convaincante parce qu'elle répond à un problème concret vécu dans les clubs sportifs, tout en offrant un très bon terrain d'apprentissage en backend, frontend et architecture logicielle.

Slide 1 - Introduction

Le problème

Pourquoi ce projet est nécessaire

Organisation actuelle dans beaucoup de clubs

  • WhatsApp pour les messages
  • Excel pour les listes et suivis
  • Appels pour confirmer les présences
  • Documents dispersés ou perdus

Conséquences

  • Planning mal suivi
  • Convocations de dernière minute
  • Informations non centralisées
  • Absence d'historique fiable

Problème central

Le club utilise trop d'outils séparés. Résultat : moins de clarté, moins d'efficacité et plus de stress dans l'organisation quotidienne.

Slide 2 - Probleme

Solution

Ce que propose EasyClubSport

Résumé

L'application permet de gérer les clubs, équipes, joueurs, profils, événements, convocations, documents et statistiques dans une interface claire et structurée.

Slide 3 - Fonctionnalites

Utilisateurs

Les rôles principaux dans l'application

01

Président

Gère le club, les équipes, les coachs, les indicateurs globaux et l'organisation générale.

02

Coach

Gère les événements, les convocations, l'effectif, les compositions et les statistiques.

03

Joueur

Consulte son planning, répond à ses disponibilités et accède à ses informations personnelles.

Role-based access Securite par policies Experience adaptee par acteur
