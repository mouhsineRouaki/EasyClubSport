EasyClubSport
Laravel API Vue 3 Tailwind CSS

Une plateforme de gestion sportive moderne, utile et réaliste

EasyClubSport est une application web qui centralise la gestion d'un club sportif : organisation, équipes, joueurs, communication, statistiques, documents et suivi quotidien.

Le problème observé

Dans beaucoup de clubs, la gestion repose encore sur WhatsApp, Excel, appels téléphoniques et documents dispersés. Cela crée des pertes de temps, des oublis et une organisation fragile.

Communication dispersée Présences mal suivies Convocations tardives

Notre réponse

  • Une application unique pour centraliser les opérations du club
  • Une gestion par rôles : président, coach, joueur, administrateur
  • Une architecture moderne : backend API + frontend séparé

Acteurs principaux

Qui utilise EasyClubSport ?

Président

  • Gère le club, les équipes et les coachs
  • Suit les cotisations, les documents et les indicateurs globaux
  • Dispose d'un tableau de bord global

Coach

  • Crée les événements sportifs
  • Gère les convocations et la composition
  • Saisit les statistiques après match

Joueur

  • Consulte son planning
  • Répond à ses disponibilités
  • Accède à ses messages, documents et statistiques

Administrateur

  • Supervise l'ensemble du système
  • Gère le support, la sécurité et les rôles globaux
  • Garantit la stabilité de la plateforme

Vue fonctionnelle

Architecture technique

Une structure moderne et professionnelle

Backend

Développé avec Laravel sous forme d'API REST.

  • Routes API dédiées
  • Validation avec FormRequest
  • Services pour la logique métier
  • Repositories pour l'accès aux données
  • Resources pour un JSON propre
  • Policies et middleware pour la sécurité

Frontend

Développé avec Vue 3 et Tailwind CSS.

  • Composants réutilisables
  • Pages métier par rôle
  • Consommation directe de l'API Laravel
  • Interface moderne, claire et responsive

Choix d'architecture

Nous avons séparé le backend et le frontend pour respecter les bonnes pratiques, faciliter la maintenance et préparer une éventuelle évolution vers mobile ou d'autres interfaces.
