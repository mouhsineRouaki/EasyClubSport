# EasyClubSport-Consomation

Projet Vue 3 minimal pour apprendre le framework depuis zero.

## Demarrage

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Structure de base

- `src/main.js` : point d entree de l application Vue
- `src/App.vue` : composant principal
- `src/style.css` : style global simple

## Ce qu on garde pour apprendre calmement

- un seul composant principal
- des variables simples dans le `<script setup>`
- affichage dans le `<template>`
- une boucle `v-for`

## Ce qu on a retire pour l instant

- appels API
- Axios
- gestion du token
- composants metier

Quand tu veux, on fait l etape suivante ensemble: comprendre `main.js`, puis `App.vue`, puis creer ton premier composant.
